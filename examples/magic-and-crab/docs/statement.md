# 魔法与螃蟹

## 题目内容

**这是一道交互题。**

出题人在某次买外卖的时候，由于没有注意商品名，买到了生螃蟹，警钟敲烂。

幸好，出题人不久后带着螃蟹转生到了一个具有魔法的世界。在这里，除了宿舍明令禁止的烹饪以外，还有一种方式------魔法，可以将生螃蟹变成可以食用的螃蟹。

在转生到魔法世界后，螃蟹获得了一个属性------「魔力」。螃蟹的「魔力」值是一个落在区间 $[2,10^{18}]$ 的正整数，记作 $x$。

使用魔法烹饪螃蟹，分为两步：

- 第一步：以一个整数 $y\in [2,100]$ 催动法术「探测术」。在运行法术后，你将得知螃蟹的「魔力」值与你给定的整数的最大公约数，即 $\gcd(x, y)$。
- 第二步：当你对螃蟹的「魔力」值有一定把握后，以一个整数 $z\in[2,100]$ 催动法术「净化术」。如果 $z$ 与 $x$ 互质，则螃蟹可以食用；否则，出现问题，螃蟹不可食用。

然而，出题人的魔法能力有限，只能使用不超过 $20$ 次「探测术」魔法。你可以帮助出题人吃到螃蟹吗？

## 交互协议

如果你希望以整数 $y$ 发动一次「探测术」，请向标准输出输出 \tt{? y}。 其中 $y$ 必须满足 $2\le y\le 100$。

当你输出这一行并刷新缓冲区后，交互器会返回一行一个整数 \tt{g}。其中 $g=\gcd(x, y)$。

如果你希望以整数 $z$ 驱动「净化术」，请向标准输出输出 \tt{! z}。其中 $z$ 必须满足 $2\le z\le 100$。

可以证明，在题目的限制之下，总存在一个 $z\in [2,100]$，使得 $x$ 与 $z$ 互质。

你可以使用下面的方法刷新缓冲区：

- C：`fflush(stdout);`
- C++：`std::cout.flush();`
- Java：`System.out.flush();`
- Python：`sys.stdout.flush()`

例如，如果你希望像样例的第一个输出一样，以数字 $8$ 催动「探测术」，请使用：

```c
// C
#include <stdio.h>
printf("? %d\n", 8);
fflush(stdout);
int g;
scanf("%d", &g);
```

```c++
// C++
#include <iostream>
std::cout << "? " << 8 << std::endl;
std::cout.flush();
int g;
std::cin >> g;
```

``` java
// Java
Scanner scanner = new Scanner(System.in);
try {
    System.out.println("? " + 8);
    System.out.flush();
    int g = scanner.nextInt();
} catch(IOException exception) {}
```

```python
# Python
import sys
print(f"? {8}")
sys.stdout.flush()
g = int(input())
```

## 样例

```input1

4

3

2

1

```

```output1
? 8

? 9

? 10

? 11

! 13

```

## 样例说明

样例中，螃蟹的「魔力」值为 $12$。

交互题与其他题目不同的是，交互题的输入是动态的，交互器会根据你的程序不同的策略输入相应的值，故 \**请不要像其他题目一样，将样例输入复制粘贴到 stdin 里**。
