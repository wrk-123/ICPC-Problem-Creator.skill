# 娄山关扫雷

## 题目描述

“娄山关”雄踞大娄山脉之前，地处贵州遵义与桐梓交界处，素有“黔北第一险隘”之称，自古便是兵家必争的咽喉要地。1935 年中央红军长征途经此地，先后两次鏖战娄山关：首战为遵义会议保驾护航，二战取得长征以来首场大捷，战后更留下“雄关漫道真如铁，而今迈步从头越”的千古名篇。

一名大学生参观娄山关战斗遗址后深受触动，结合经典扫雷玩法开发了一款娄山关主题推演小游戏。

给定一个 `n x n` 的地雷图 `A`。对地雷图上的每一个单元格，要么为空格，要么为地雷。按照经典扫雷规则，每个空格会标注一个数字，表示它周围 `8` 个相邻格子中地雷的总数。记 `S(A)` 为地雷图 `A` 上所有空格标注数字之和。

接下来把 `A` 中所有地雷格和空格互换，得到新的地雷图 `Ā`。也就是说，原来是地雷的位置会变为空格，原来是空格的位置会变为地雷。类似地，记 `S(Ā)` 为 `Ā` 上所有空格标注数字之和。

初始时整张地雷图 `A` 全为空格。现在你需要恰好在图中放置 `k^2` 枚地雷。每个格子至多放置一枚地雷，并且必须放完这 `k^2` 枚地雷。请最小化

`|S(A) - S(Ā)| + S(A) + S(Ā)`

并输出这个最小值。

## 输入格式

输入一行两个整数 `n, k`，满足 `1 <= n <= 50`，`0 <= k <= n`。

其中地雷图一共有 `n` 行 `n` 列，需要恰好放置 `k^2` 枚地雷。

## 输出格式

输出一个整数，表示题目中表达式可以取得的最小值。

## 样例

```input1
4 2
```

```output1
18
```

## 样例说明

将 `4` 枚地雷放成左上角的一个 `2 x 2` 方块即可达到最优。此时跨越“地雷格-空格”边界的相邻格对一共有 `9` 对，因此 `S(A) = S(Ā) = 9`，答案为 `18`。
