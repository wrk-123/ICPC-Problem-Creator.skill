# 娄山关扫雷题解

## 核心化简

把棋盘看成一个图：每个格子是一个点，八连通相邻的两个格子之间连一条边。

如果一条边的两个端点一个是地雷、一个是空格，那么：

- 在 `A` 中，这条边会恰好给那个空格贡献 `1`；
- 在 `Ā` 中，地雷和空格互换后，这条边仍然会恰好给那个空格贡献 `1`。

因此对于任意布雷方案 `A`，都有：

- `S(A)` 等于“地雷格与空格之间的相邻边数”；
- `S(Ā)` 也等于同一个值。

设这个值为 `cut(A)`，则原式变成：

`|S(A) - S(Ā)| + S(A) + S(Ā) = 0 + 2 * cut(A)`。

所以题目等价于：

> 在 `n x n` 的八连通棋盘上，选出恰好 `k^2` 个格子作为地雷，使地雷集合与其补集之间的割边数最小。

最终答案就是最小割边数的 `2` 倍。

## 形状压缩

设某一行里有若干个地雷。把这一行的所有地雷整体向左挤，不会让它与上一行、下一行以及本行内部产生更多的“地雷-空格”相邻对；直观上说，越分散，边界越长，越靠左聚拢，边界越短。

对所有行都做这样的“左压缩”后，再把行长按不增顺序从上到下排列，同样不会让边界变差。于是总能找到一个最优解，它满足：

- 每一行的地雷都连续地放在最左侧；
- 各行地雷个数 `l1, l2, ..., ln` 满足 `l1 >= l2 >= ... >= ln >= 0`；
- `l1 + l2 + ... + ln = k^2`。

这就是一个贴着左上角的 Ferrers 图形。接下来只需要在所有这样的行长序列里找最优值。

## 如何计算一个形状的割边数

### 1. 点度数贡献

如果某个格子被选为地雷，那么它对割边数的贡献可以从

`cut = 所有选中点的度数和 - 2 * 选中点之间的内部边数`

这个公式来算。

在八连通棋盘中，角上的点度数为 `3`，边上的点度数为 `5`，内部点度数为 `8`。预处理每一行前缀 `degPrefix[row][len]`，表示这一行最左边 `len` 个格子的度数和。

### 2. 同一行内部边数

一行长度为 `x`，同一行内部的水平边有 `x - 1` 条。

### 3. 相邻两行之间的内部边数

若上一行长度为 `a`，当前行长度为 `b`，并且 `a >= b`，那么两行之间的内部边数只与 `a, b` 有关：

- 竖直边有 `b` 条；
- 右下方向的对角边有 `b - 1` 条；
- 左下方向的对角边：
  - 若 `a = b`，只有 `b - 1` 条；
  - 若 `a > b`，有 `b` 条。

因此

- `cross(a, b) = 0`，当 `b = 0`；
- `cross(a, b) = 3b - 2`，当 `a = b > 0`；
- `cross(a, b) = 3b - 1`，当 `a > b > 0`。

## 动态规划

定义 `dp[row][used][prev]`：

- 已经处理完前 `row` 行；
- 一共放了 `used` 枚地雷；
- 第 `row` 行（也就是上一行）的长度为 `prev`；
- 此时能够得到的最小割边数。

枚举当前行长度 `cur`，满足：

- `0 <= cur <= prev`；
- `used + cur <= k^2`。

转移增量为：

- 当前行最左 `cur` 个格子的度数和；
- 减去当前行内部水平边的两倍；
- 若不是第一行，再减去它与上一行内部边数的两倍。

也就是：

`add = degPrefix[row][cur] - 2 * (cur - 1) - 2 * cross(prev, cur)`

其中第一行没有 `cross` 项，且当 `cur = 0` 时水平边贡献视为 `0`。

最终答案是

`2 * min(dp[n][k^2][prev])`。

## 正确性说明

### 引理 1

对任意布雷方案 `A`，都有 `S(A) = S(Ā) = cut(A)`。

证明见“核心化简”部分：每一条连接“地雷格”和“空格”的相邻边，会在 `A` 和 `Ā` 中各贡献一次，且不会重复或遗漏。

### 引理 2

总存在一个最优方案，其地雷格组成一个左上角对齐、行长不增的 Ferrers 图形。

证明要点如下：

- 固定其它行时，把某一行中的地雷向左压缩，只会让它与本行、上一行、下一行的交界更集中，不会增加“地雷-空格”相邻对数；
- 对所有行左压缩后，再按行长从大到小摆在上方，本质上是在继续做同类压缩，同样不会使边界变长。

因此最优解一定可以在这类形状中找到。

### 引理 3

对于任意一个 Ferrers 图形，题解中的计数公式能够准确给出其割边数。

证明：

- 每个被选中的格子先按自身度数计入一次；
- 每条内部边会在两个端点的度数和中被算了两次，所以需要减去 `2` 倍内部边数；
- 内部边只可能出现在同一行内部，或两条相邻行之间；
- 对这两类边分别计数，恰好得到题解中的公式。

因此公式正确。

### 引理 4

`dp` 的状态定义与转移可以枚举到所有合法 Ferrers 图形，并且不会漏掉最优解。

证明：

- Ferrers 图形恰好对应一组行长不增序列；
- `prev` 保证当前行长度不超过上一行；
- `used` 保证总面积恰好为 `k^2`；
- 每次转移准确加入当前这一行新增的割边贡献。

所以 `dp` 精确覆盖了全部候选形状，取最小值即得到最优割边数。

### 定理

算法输出的答案等于题目要求的最小值。

由引理 2，最优解一定可以限制在 Ferrers 图形中；由引理 3 和引理 4，`dp` 能正确求出这些图形中的最小割边数；再由引理 1，题目原式等于该最小割边数的两倍。因此算法正确。

## 复杂度分析

设 `m = k^2`。

- 状态数为 `O(n * m * n)`；
- 每个状态再枚举行长 `cur`，总复杂度为 `O(n^2 * m)`，由于 `m <= n^2`，也可写成 `O(n^4)`；
- 空间复杂度为滚动数组下的 `O(m * n)`。

在 `n <= 50` 的范围内完全可行。
